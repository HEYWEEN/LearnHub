# LearnHub - 智能在线学习平台

> 南京大学软件学院《互联网计算》课程大作业

## 🎯 项目简介

LearnHub 是一个现代化的在线学习平台，提供课程学习、进度跟踪和智能问答等功能。采用前后端分离架构，使用 React + Node.js + MongoDB 技术栈。

## ✨ 核心功能

### 用户系统
- **用户注册登录** - JWT token认证
- **个人信息管理** - 头像上传、资料编辑
- **权限管理** - 学生、教师、管理员角色

### 课程系统
- **课程浏览** - 分页显示、分类筛选
- **课程搜索** - 关键词搜索、高级筛选
- **课程详情** - 完整信息展示、评价系统

### 学习功能
- **视频学习** - 在线播放、进度控制
- **学习进度** - 自动保存、进度追踪
- **笔记系统** - 实时记录、关联时间点

### AI助手
- **智能问答** - 集成OpenAI API解答问题
- **学习建议** - 个性化课程推荐

### 管理功能
- **课程管理** - 课程CRUD、内容管理
- **用户管理** - 用户信息、权限管理
- **数据统计** - 学习数据可视化

## 🛠 技术栈

### 前端技术
- **框架**: React 18 + Vite
- **UI库**: Ant Design 5.x
- **状态管理**: Redux Toolkit
- **路由**: React Router v6
- **图表**: Recharts
- **HTTP**: Axios

### 后端技术
- **运行时**: Node.js
- **框架**: Express.js
- **数据库**: MongoDB + Mongoose
- **认证**: JWT
- **文件上传**: Multer
- **实时通信**: Socket.io

### 开发工具
- **版本控制**: Git
- **包管理**: npm
- **代码检查**: ESLint
- **构建工具**: Vite
